# phphelper
An helper method collections based on laravel which can be used in any php project

## How to use it ?
create a composer.json in your root project folder. Then add this code

```
{
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/fftfaisal/phphelper"
        }
    ]
}
```
